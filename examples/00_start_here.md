# Welcome to the mde Tutorial

**mde** is a terminal Markdown editor that lives entirely in your terminal.
No mouse, no menus — just your keyboard, your text, and two views of the
same document.

> **You are reading this in preview mode.** Press `Ctrl+P` to flip into
> edit mode and see the raw Markdown underneath. Press `Ctrl+P` (or
> `Escape`) to return here.

## The two modes at a glance

**Preview mode** (where you are now)

- Markdown is rendered: headings are coloured, delimiters are hidden,
  links are numbered `(N)`.
- Read-only — you cannot type here.
- Navigate with arrow keys, `Page Up/Down`, `Home/End`.

**Edit mode** (press `Ctrl+P`)

- Raw Markdown is shown; delimiters are dimmed so content stands out.
- Full cursor movement and text editing.
- Press `Escape` or `Ctrl+P` to return to preview.

## Essential keys to know right now

| Key | What it does |
|-----|-------------|
| `Ctrl+P` | Toggle between preview and edit mode |
| `Ctrl+O` | Open a different file |
| `Ctrl+T` | Open the Table of Contents |
| `Ctrl+L` | Follow a link by its number |
| `Ctrl+Q` | Quit (press twice if there are unsaved changes) |
| `F1` | Show the built-in help |
